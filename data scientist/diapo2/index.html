Missions

Les missions d'un data scientist sont multiples :
- Analiser les données (data)
- Prendre des décisions marketing à partir des données
- Définir des solutions de stockage pour les données
- Construirent des algorithmes de recherche
- création de modèles de prédictions
- Création de tableaux de bord pour l'exploitation des données
- Veille technologique
- Machine learning
- Gestion d'intelligence artificiel
- Algorithmes d'automatisation
- System de reconnaissance
